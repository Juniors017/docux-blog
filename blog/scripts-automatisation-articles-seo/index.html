🎯 Le Problème : La Création Manuelle d'Articles​

🎨 Snippets VS Code : L'Intégration Native​

Installation Automatique​

Les snippets sont pré-configurés dans .vscode/docusaurus-snippets.json

Utilisation Intuitive​

  1. Créer un nouveau fichier .mdx
  2. Taper doc- puis Tab
  3. Choisir le template :
  • doc-blog → BlogPosting
  • doc-tech → TechArticle
  • doc-howto → HowTo Guide
  1. Naviguer avec Tab entre les champs
  2. Variables automatiques : date, année, etc.

🐛 Dépannage​

Erreurs Communes​

"Directory already exists"

# Solution : Utiliser un slug différent ou supprimer le dossier existant
rm -rf blog/2025/08/29/mon-article/

"Invalid schema type"

# Types valides : BlogPosting, TechArticle, HowTo, FAQPage, CollectionPage, 
# SoftwareApplication, Course, WebPage, AboutPage, ItemListPage
node scripts/quick-article.js TechArticle "Mon Guide"

"Permission denied"

# Solution : Vérifier les permissions d'écriture
chmod 755 blog/

Debug Mode​

Décommentez les lignes console.log dans les scripts pour activer le mode debug détaillé.
