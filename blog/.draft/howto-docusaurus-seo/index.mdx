# Comment Optimiser le SEO de votre Site Docusaurus - Guide Complet

:::tip Prérequis

- ✅ Site Docusaurus existant ou nouveau
- ✅ Connaissances de base en React/JavaScript
- ✅ Node.js 18+ installé
- ✅ 2 heures de temps libre
  :::

## 🎯 Objectif du Guide

À la fin de ce tutoriel, votre site Docusaurus aura :

- **Schémas JSON-LD** automatiques pour tous les contenus
- **Rich Results Google** : breadcrumbs, articles, collections
- **Métadonnées optimisées** : OpenGraph, Twitter Cards
- **Architecture SEO professionnelle** avec composants modulaires

## 📋 Étape 1 : Installation des Composants SEO (30 min)

### 1.1 Création du Composant SEO Principal

Créez le fichier `src/components/Seo/index.jsx` :

```jsx title="src/components/Seo/index.jsx"
import React from "react";
import Head from "@docusaurus/Head";
import { useLocation } from "@docusaurus/router";
import useDocusaurusContext from "@docusaurus/useDocusaurusContext";

export default function Seo() {
  const { siteConfig } = useDocusaurusContext();
  const location = useLocation();

  // Génération automatique des métadonnées
  const pageTitle = "Votre Site Docusaurus";
  const pageDescription = "Description par défaut";

  return (
    <Head>
      <title>{pageTitle}</title>
      <meta name="description" content={pageDescription} />
      <meta property="og:title" content={pageTitle} />
      <meta property="og:description" content={pageDescription} />
      <script type="application/ld+json">
        {JSON.stringify({
          "@context": "https://schema.org",
          "@type": "WebPage",
          name: pageTitle,
          description: pageDescription,
        })}
      </script>
    </Head>
  );
}
```

### 1.2 Intégration Global Layout

Modifiez `src/theme/Layout/index.js` :

```jsx title="src/theme/Layout/index.js"
import React from "react";
import Layout from "@theme-original/Layout";
import Seo from "@site/src/components/Seo";

export default function LayoutWrapper(props) {
  return (
    <>
      <Seo />
      <Layout {...props} />
    </>
  );
}
```

## 📋 Étape 2 : Configuration des Schémas Multiples (45 min)

### 2.1 Support des SchemaTypes Explicites

Ajoutez dans vos articles :

```yaml title="Frontmatter d'article"
---
title: "Mon Article"
schemaTypes: ["TechArticle", "BlogPosting"] # Schémas multiples
proficiencyLevel: "Beginner"
programmingLanguage: "JavaScript"
---
```

### 2.2 Détection Automatique Intelligente

Le composant détectera automatiquement :

- **TechArticle** : Mots-clés techniques (JavaScript, React, API...)
- **HowTo** : Titres avec "Comment", "Guide", structure étapes
- **BlogPosting** : Contexte blog avec auteurs

## 📋 Étape 3 : Optimisation Rich Results (30 min)

### 3.1 Configuration BreadcrumbList

Les breadcrumbs sont générés automatiquement :

- **2 niveaux** : Pages générales
