<Tooltip model="teacher" text="Docker">

Docker is a tool that helps developers build, share,
 and run applications using containers.

  </Tooltip>


### Variantes de l'exemple "Docker"

#### 1) `teacher` (image + titre automatique)

<Tooltip model="teacher" text="Docker">
Docker is a tool that helps developers build, share, and run applications using containers.
</Tooltip>

#### 2) `teacher` avec `imageAlt` et image plus petite

<Tooltip model={{ ...({}), imageAlt: 'Turtle teacher', imageSize: '48px' }} text="Docker">
Docker is a tool that helps developers build, share, and run applications using containers.
</Tooltip>

#### 3) Position à droite

<Tooltip model="teacher" text="Docker" position="top">
Docker is a tool that helps developers
 build, share, and run 
 applications using containers.
</Tooltip>

#### 4) Sans image (modèle custom)

<Tooltip model={{ backgroundColor: '#222', color: '#fff', padding: '10px' }} text="Docker">
Docker is a tool that helps developers build, share, and run applications using containers.
</Tooltip>

#### 5) Contenu riche via `content` (JSX)

<Tooltip model="teacher" text="Docker" content={<div><strong>Docker</strong><p>Docker helps build, ship and run apps.</p></div>}>
  Trigger ignored when `content` used (text still trigger)
</Tooltip>




## 1) Modèle prédéfini (clé)


<Tooltip model="info" text="Info : ceci est un tooltip">
  Passez la souris sur moi
</Tooltip>


## 2) Modèle personnalisé par objet


<Tooltip model={{ backgroundColor: '#222', color: '#fff', padding: '10px' }} text="Tooltip sombre">
  <span>Survolez ce span</span>
</Tooltip>


## 3) Modèle via fonction (transformation)


<Tooltip model={(base) => ({ ...base, padding: '12px', fontWeight: '600' })} text="Transformé">
  Composant enfant
</Tooltip>


## 4) Contenu riche via `content` (JSX)


<Tooltip content={<div><strong>Important</strong><div>Plus d'info</div></div>}>
  Texte riche
</Tooltip>


## 5) Texte nu (MDX mettra le wrapper dans un `<p>`, le Tooltip gère le wrapper interne)


<Tooltip text="Tooltip sur texte nu">
  Ceci est un texte simple
</Tooltip>
